# Anim AI System

Generative AI tool for creating animations from text and image prompts using diffusion models.

## Features

- Text-to-animation generation
- Image-to-animation conversion
- Configurable frame rate and resolution
- Style transfer support

## Tech Stack

Python, Diffusion Models, GANs

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Anim-AI-System.git
cd Anim-AI-System
pip install -r requirements.txt
```

### Usage

```bash
python generate.py --prompt 'a cat walking'
```
